Lista con puntos

  • Se empieza con ul
  • Cada elemento se escribe entre li[...]/li
  • Se cierra con /ul

Lista ordenada

  1. Se empieza con ol
  2. Cada elemento se escribe entre li[...]/li
  3. Se cierra con /ol
