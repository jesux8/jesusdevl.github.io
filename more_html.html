Enlace desde texto (J.A.)

La etiqueta para poner enlaces es "a". Dentro de la etiqueta que abre "a", debemos escribir href= y el enlace a donde queremos que nos lleve, y después escribiremos el texto donde tendremos que pinchar.

Por ejemplo, si pinchas aquí te llevará a la página del instiuto.

Enlace absoluto

Es un enlace completo a una página web.Debe incluir la palabra inicial http:// seguida del nombre del dominio. Por ejemplo, https://www.youtube.com/


Enlace relativo

Esto quiere decir que el enlace abrirá una ubicación de nuestro sitio web. Por ejemplo, si pinchas aquí te llevará de nuevo al índice de la página.


Enalce que se abra en una ventana nueva (con el atributo target)

Esto quiere decir que el enlace se abrirá en una pestaña nueva. Se consigue escribiendo target="_blank" dentro de la etiqueta "a".

Los dos enlaces de ejemplo anteriores están puestos así, por eso se abren en una nueva pestaña.

Tablas

Con la etiqueta "table" podemos crear tablas. Una tabla debe comenzar con una fila de encabezado con los nombres de las columnas. El elemento thead inicia el área de encabezado, el elemento tr crea una fila y los elementos th crean celdas en la fila. Una tabla puede constar de cualquier número de filas de datos después del encabezado. La etiqueta tbody comienza el área del cuerpo (datos), la misma etiqueta tr crea una fila y las etiquetas crean celdas de datos en cada fila.

Ejemplo: Un horario

Hora Lunes Martes
8:00 Matemáticas Lengua
10:00 Física Inglés
12:00 TIC Historia
